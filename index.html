🔑 Anthropic API Key

Loreforge calls Anthropic's API directly from your browser. Your key is needed to generate worlds, simulations, and adventures.

Your key is private. It's stored only in your browser's localStorage. It is never sent to any server except Anthropic. Other users of this site cannot see or use it.

Don't have one? Get a key at console.anthropic.com.
